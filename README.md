# 🎮 CCG Bot

Discord bot for League of Legends built with Riot Games API.

## ⚡ Features

- 📊 Player match statistics
- 🎮 Match details breakdown
- 👀 Live game tracking
- 📋 Player list tracking
- 🎲 5Stars role randomizer game
- 🔔 Match notifications in Discord channel
- 🏓 help commands

---

## 🧠 Tech Stack

- Python 3.11+
- discord.py (slash commands)
- Riot Games API
- aiohttp (async requests)
