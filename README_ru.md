# CCG Bot

Discord-бот для отслеживания матчей в League of Legends через Riot API. Проект собирает статистику по отслеживаемым игрокам, сохраняет матчи в SQLite, показывает детальную аналитику по командам и автоматически отправляет уведомления о новых играх в Discord.

## О проекте

- slash-команды на `discord.py`
- интеграция с `Riot Games API`
- локальное хранение матчей и статистики в `SQLite`
- фоновый watcher для проверки новых матчей
- отдельная игровая мини-механика для Discord-сервера

## Основные возможности

- Просмотр статистики игрока по последним 100 матчам через `/lolstats`
- Детальный разбор конкретного матча через `/matchdetails`
- Проверка, кто из отслеживаемых игроков сейчас в игре через `/whoingame`
- Автоматическое отслеживание новых матчей и отправка embed-уведомлений в Discord
- Хранение списка отслеживаемых игроков и whitelist пользователей для управляющих команд
- Расчет рекордов по сохраненным матчам через `/records`
- Мини-игра `/5stars` для случайной раздачи ролей с учетом нежелательной роли игрока
- Встроенные utility-команды для проверки бота и отображения справки

## Команды

| Команда                                        | Описание                                                                             |
| ---------------------------------------------- | ------------------------------------------------------------------------------------ |
| `/lolstats nickname match_count result_filter` | Показывает статистику по последним матчам игрока с фильтрацией по победам/поражениям |
| `/matchdetails nickname match_number`          | Показывает подробный разбор выбранного матча из истории игрока                       |
| `/whoingame`                                   | Проверяет, кто из отслеживаемых игроков сейчас находится в активной игре             |
| `/playerslol`                                  | Выводит список отслеживаемых игроков                                                 |
| `/addplayer nickname`                          | Добавляет игрока в отслеживание                                                      |
| `/removeplayer nickname`                       | Удаляет игрока из отслеживания                                                       |
| `/allowuser user`                              | Добавляет пользователя в whitelist на управляющие команды                            |
| `/disallowuser user`                           | Удаляет пользователя из whitelist                                                    |
| `/setchannel`                                  | Назначает текущий канал каналом для матч-уведомлений                                 |
| `/records`                                     | Показывает лучшие рекорды отслеживаемых игроков по сохраненным матчам                |
| `/5stars`                                      | Запускает мини-игру на 5 игроков с рандомной раздачей ролей                          |
| `/ping`                                        | Проверка доступности бота                                                            |
| `/helplol`                                     | Справка по командам                                                                  |

## Архитектура

Проект разбит на несколько логических слоев:

- `bot.py` инициализирует Discord-бота, Riot API клиент, базу данных и подключает расширения
- `cogs/` содержит пользовательские slash-команды
- `services/` содержит фоновые процессы, которые работают независимо от ручных команд
- `riot/riot_api.py` инкапсулирует работу с Riot API
- `db/` отвечает за схему, подключение к SQLite и repository-слой
- `utils/` содержит файловое хранилище, хелперы и декларативное описание рекордов
- `data/` хранит конфиги, списки игроков, whitelist и локальную базу

### Что хранится в базе

- `players` — отслеживаемые игроки и их Riot ID
- `matches` — сохраненные матчи и сырой JSON от Riot API
- `player_match_stats` — агрегированная статистика отслеживаемых игроков по матчам

## Технологии

- Python 3.11+
- `discord.py`
- `aiohttp`
- `aiosqlite`
- `python-dotenv`
- Riot Games API
- SQLite

## Как запустить локально

### 1. Установить зависимости

```bash
pip install discord.py aiohttp aiosqlite python-dotenv requests flask
```

### 2. Создать `.env`

```env
DISCORD_TOKEN=your_discord_bot_token
RIOT_API_KEY=your_riot_api_key
```

### 3. Запустить бота

```bash
python bot.py
```

При первом запуске бот:

- подключится к Discord
- создаст или инициализирует SQLite-базу по `db/schema.sql`
- загрузит список чемпионов
- подключит slash-команды и фоновый watcher

## Важные файлы данных

- `data/players.txt` — список отслеживаемых игроков по `puuid`
- `data/allowed_users.txt` — whitelist Discord-пользователей для управляющих команд
- `data/config.json` — конфиг с `notify_channel_id`
- `data/riftwatcher.db` — локальная база матчей и статистики
